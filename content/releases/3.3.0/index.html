Release Notes

This release contains some bug fixes, improvements, and a new charset.

Bug Fixes

  • [DAFFODIL-2267] Warnings emmited on pre-compiled parsers
  • [DAFFODIL-2575] dfdlx:trace function should not write directly to stderr
  • [DAFFODIL-2613] Install with DNF must install "apache-daffodil" rather than "daffodil"
  • [DAFFODIL-2619] Add InfosetInputter with minimal overhead
  • [DAFFODIL-2623] Fix C compilation errors due to name collisions
  • [DAFFODIL-2627] Performance regression in TDML processor
  • [DAFFODIL-2629] Runtime SDE with SetVariable expression
  • [DAFFODIL-2630] nullPointerException when using TDML runner.trace
  • [DAFFODIL-2635] dfdl:valueLength on child element returns unexpected Result
  • [DAFFODIL-2641] Incorrect scaladoc in sapi/japi for isError
  • [DAFFODIL-2642] Daffodil Resolver - schemaLocation not found should be SDE
  • [DAFFODIL-2648] fn:exists fails to unparse when element does not exist
  • [DAFFODIL-2650] using config file with cli parse or save parser causes backtrace
  • [DAFFODIL-2652] Ability to disable all alignment
  • [DAFFODIL-2653] Usage error: DirectOrBufferedDataOutputStream.this.isFinished.unary_! stack trace on unparsing
  • [DAFFODIL-2655] Modify release candidate container to work for both daffodil and daffodil-vscode
  • [DAFFODIL-2656] prefixed length element has no dfdl:contentLength nor dfdl:valueLength
  • [DAFFODIL-2657] prefixedLengthType which itself has lengthKind 'implict' fails with None.get
  • [DAFFODIL-2662] P8 cyclic deadlock regression
  • [DAFFODIL-2664] PNG DFDL Schema fails regression testing on 3.3.0-SNAPSHOT
  • [DAFFODIL-2665] quality-gate icon/badge doesn't work on wiki page
  • [DAFFODIL-2666] NITF cyclic deadlock regression
  • [DAFFODIL-2673] enum no-match on unparsing causes Invariant Broken abort
  • [DAFFODIL-2674] sonarcloud and other minor cleanup

New Features

  • [DAFFODIL-2659] Special 6-bit character set needed

Improvements

  • [DAFFODIL-2595] Create podman/docker container for maintaining daffodil-site
  • [DAFFODIL-2604] setup for 3.3.0-SNAPSHOT development on main branch
  • [DAFFODIL-2622] Fix accumulated solarcloud warnings
  • [DAFFODIL-2677] Prepare for Release 3.3.0

Deprecation/Compatibility

There are no deprecations. This release is fully compatible with all functionality of the prior release.

Dependency Changes

The following dependencies have been added or updated:

  • FasterXML Jackson Core 2.13.2 (update)
  • FasterXML Woodstox Core 6.2.8 (update)
  • Log4j api 2.17.2 (update)
  • Log4j core 2.17.2 (update)
  • Scala Parser Combinators 2.1.1 (update)
  • Typesafe Config 1.4.2 (update)
  • Xerces 2.12.2 (update)

Code Generator

  • OS-Lib 0.8.1 (update)

Command Line Interface

  • Log4j core 2.17.2 (update)

Schematron Validator

  • Saxon-HE 11.2 (update)

Test

  • Junit-interface 0.13.3 (update)
